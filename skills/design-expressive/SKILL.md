# Design: Expressive

Apply to: **$ARGUMENTS**. Foundation rules from `design-dna` apply; this skill sets the dial positions.

**Reference world:** a great magazine cover, a concert poster, a title sequence.
The goal is a page someone screenshots.
Expressive design is loud on purpose, in one place at a time; chaos everywhere is as forgettable as minimal-gray everywhere.

**The one big idea:** before styling anything, decide the page's single memorable move - the thing a visitor describes later (a headline cropping off-canvas, type interacting with a cutout image, a violent color flip between sections).
ONE big idea executed flawlessly, supported by everything else being quietly excellent; two compete, five are noise.

Set every dial from [dials.md](dials.md): typography, color, layout, motion, and the never list.

**Self-check:** screenshot test (is there a frame someone would post?).
Memory test: name the one big idea in five words.
Calm test: is the body reading layer still disciplined underneath the show?
