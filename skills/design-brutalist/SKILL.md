# Design: Brutalist

Apply to: **$ARGUMENTS**. Foundation rules from `design-dna` apply; this skill sets the dial positions.

**Reference world:** béton brut architecture, early-web honesty, a screen-printed gig flyer, a terminal.
Brutalism shows the structure instead of hiding it - but crafted rawness: every "ugly" choice is exact.
The difference between brutalist and broken is alignment.

Set every dial from [dials.md](dials.md): typography, color, structure, components, motion, and the never list.

**Self-check:** structure test (can you see the grid without devtools?).
Honesty test: is anything decorated to look like something it isn't?
Precision test: are all the hard edges aligned to the pixel?
