# Design: Minimal

Apply to: **$ARGUMENTS**. Foundation rules from `design-dna` apply; this skill sets the dial positions.

**Reference world:** Rams-era Braun, a Kenya Hara catalog, Swiss typography.
Minimalism is design with nowhere to hide: every remaining element must be exactly placed, because it is all the viewer has.

**The discipline:** for every element ask what breaks if it is removed; nothing -> remove it.
This applies to borders, icons, labels repeating the obvious, decorative images, second buttons, taglines.
Fewer, better - not gray-on-white versions of the same clutter.

Set every dial from [dials.md](dials.md): typography, color, space, components, motion, and the never list.

**Self-check:** remove-one-more test (delete one more element; if nothing breaks, you weren't done).
Squint: exactly one focal point.
Alignment audit at 200% zoom: no edge off its spine.
